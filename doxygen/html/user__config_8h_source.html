130/*
131 ****************************************************************************************
132 *
133 * Advertising or scan response data for the following cases:
134 *
135 * - ADV_IND: Connectable undirected advertising event.
136 * - The maximum length of the user defined advertising data shall be 28 bytes.
137 * - The Flags data type are written by the related ROM function, hence the user shall
138 * not include them in the advertising data. The related ROM function adds 3 bytes in
139 * the start of the advertising data that are to be transmitted over the air.
140 * - The maximum length of the user defined response data shall be 31 bytes.
141 *
142 * - ADV_NONCONN_IND: Non-connectable undirected advertising event.
143 * - The maximum length of the user defined advertising data shall be 31 bytes.
144 * - The Flags data type may be omitted, hence the user can use all the 31 bytes for
145 * data.
146 * - The scan response data shall be empty.
147 *
148 * - ADV_SCAN_IND: Scannable undirected advertising event.
149 * - The maximum length of the user defined advertising data shall be 31 bytes.
150 * - The Flags data type may be omitted, hence the user can use all the 31 bytes for
151 * data.
152 * - The maximum length of the user defined response data shall be 31 bytes.
153 ****************************************************************************************
154 */

161 "\xEE\x8F\x7A\x37\xCC\x56\x5D\xAA\xED\x40\xA0\xF0\xC2\x94\xE0\xD6")
